# DevRank API

A backend REST API platform where developers build verifiable public profiles 
through peer-reviewed projects — designed for hiring teams to assess real coding ability.

## Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- SQLite + Sequelize

## Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` with `MONGO_URI` and `PORT`
4. Run `npm run dev`
